# ProCore 官網 SEO 群集機會盤點

## 目前案例庫概況

依目前根目錄 `article-*.html` 檔名粗略盤點：

- 案例文章：約 86 篇
- 鑰匙遺失 / 全丟相關：約 38 篇
- 品牌量較多：BMW 13、Toyota 9、VW 8、Benz 6
- 地區量較多：台中 6、和美 5、雲林 4、林口 4、花壇 3、彰化 3、高雄 3

## 最值得優先做的 SEO 聚合頁

### 1. Toyota Altis 鑰匙服務頁

理由：Toyota 案例已有 9 篇，其中 Altis 有台中龍井、員林、清水等案例。Altis 搜尋量大，適合作為平價高量車款入口。

建議頁面：

- `toyota-altis-car-key.html` 或服務/車款頁路徑
- 主題：Toyota Altis 智能鑰匙、鑰匙遺失、備用鑰匙、現場處理評估
- 內部連結：Altis 龍井、Altis 員林、Altis 清水案例

### 2. 台中汽車鑰匙地區頁

理由：台中案例量最高，且龍井、清水、北屯、南屯、豐原等可形成地區群集。

建議頁面：

- `taichung-car-key.html`
- 主題：台中汽車鑰匙、智能鑰匙、鑰匙遺失、備用鑰匙、到場評估
- 內部連結：台中各行政區案例與服務範圍

### 3. BMW 智能鑰匙 / 鑰匙全丟頁

理由：BMW 案例最多，且高單價、高信任需求，適合獨立商業頁。

建議頁面：

- `bmw-smart-key-service.html`
- 主題：BMW 智能鑰匙、鑰匙遺失、現場車、拍場車、備用鑰匙
- 注意：不公開防盜與操作流程

### 4. VW / VAG 智能鑰匙頁

理由：VW 文章與 VAG 技術類文章已有基礎，可承接 Golf、T5、T6、Arteon 等搜尋。

建議頁面：

- `vw-car-key-service.html`
- 主題：VW Golf 智能鑰匙、VAG 車系鑰匙服務、備用鑰匙與異常評估

### 5. 智能鑰匙全丟服務頁

理由：全丟是高意圖、高急迫性的案件類型，但頁面需安全表述。

建議頁面：

- `smart-key-lost-service.html`
- 主題：遇到智能鑰匙遺失時該準備什麼、是否需要拖吊依車況判斷、現場照片的重要性
- CTA：車款、年份、所在地、問題、照片

## 內容規則

聚合頁可以比案例文更商業，但仍需遵守：

- 不寫可複製技術流程
- 不寫設備名稱、PIN、EEPROM、OBD、bypass 等內容
- 不保證免拖吊或一定可到場處理
- 不寫未確認的遙控/感應/啟動全部正常
- CTA 聚焦準備資料與 LINE/電話，不誇張承諾
