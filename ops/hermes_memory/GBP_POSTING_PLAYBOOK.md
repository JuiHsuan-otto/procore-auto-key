# ProCore Google 商家貼文 Playbook

## 目標

Google 商家貼文的任務不是塞滿關鍵字，而是讓在地搜尋者看到 ProCore 正在處理真實車款案件，並透過商家檔案既有電話、按鈕或連結採取下一步。

## 正文禁止放入

Google 商家貼文正文不要放：

- `LINE：@420gknem`
- `電話：0909277670`
- 完整案例 URL 或 `https://www.carkey.com.tw/...`
- `完整案例：...`

聯絡方式與連結應放在 Google 商家檔案既有欄位、CTA 按鈕或後台連結欄位，不寫進正文。

## 降低被拒風險的寫法

Google 對汽車鑰匙、防盜、全丟、無法啟動、解鎖等詞可能比較敏感。商家貼文建議比官網文章更保守：

- 用「汽車鑰匙服務案例」取代過度直接的「鑰匙全丟」
- 用「智能鑰匙相關狀況」取代「智能鑰匙全丟」
- 用「依車款年份與現場條件協助評估處理」取代技術流程
- 不寫設備、操作方式、車輛防盜流程
- 不寫「沒有鑰匙也能處理」「免拖吊」這類容易被審核誤判的句子

## 建議結構

```text
地區 + 車款 + 汽車鑰匙服務案例。

車主遇到智能鑰匙相關狀況，ProCore 依車款年份、車輛狀態與現場條件協助評估處理。遇到鑰匙遺失、備用鑰匙需求或智能鑰匙無法正常使用，可以先準備車款、年份、所在地與現場照片，方便判斷後續安排。

公開案例內容只分享地區、車款與服務情境，不揭露車主個資、完整地址或可複製的技術流程。
```

## 圖片建議

優先使用：

- 車身或現場遠景
- 無可讀車牌、臉、地址、文件、VIN
- 無工具、設備細節、可複製鑰匙齒紋
- 畫面乾淨，不像技術操作教學

若 Google 商家拒登，先嘗試：

1. 移除正文中的全丟、拖吊、無法使用、解鎖等敏感詞。
2. 不在正文放 URL、電話、LINE。
3. 更換更保守的車輛外觀或品牌形象圖。
4. 使用 CTA 按鈕放官網連結，不在正文重複。

## 後台欄位策略

- 正文：只放案例摘要與準備資料。
- CTA：使用「瞭解詳情」連到官網案例或服務頁。
- 電話：使用 Google 商家檔案既有電話，不寫入正文。
- LINE：不要寫正文；可由官網頁面承接。
